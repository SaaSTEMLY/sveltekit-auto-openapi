# Milestones

## Current Status

> **⚠️ v0 - Experimental**
>
> This plugin is in early development (v0) and **not recommended for production use**. APIs will most probably change, and there may be undiscovered bugs. Use at your own risk.
>
> **Contributions are welcome!** If you'd like to help improve this project, please feel free to open issues or submit pull requests.

## Roadmap for Version 0

The following milestones need to be completed before releasing v1:

- [ ] **Explore edge cases and find errors** - Thorough testing across different SvelteKit configurations and route patterns to identify and fix edge cases
- [ ] **Optimize Vite plugin** - Improve build performance, reduce overhead, and optimize the virtual module generation process
- [ ] **Expand docs and create a docs website** - Complete comprehensive documentation with examples, guides, and API references
- [ ] **Add proper tests** - Implement comprehensive test coverage for all plugin functionality, validation, and type generation
- [ ] **Publish V1** - Release stable v1.0.0 with finalized APIs and production-ready quality
